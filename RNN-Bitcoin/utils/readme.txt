Le dossier utils/ est utilisé pour stocker des fichiers Python contenant des fonctions réutilisables dans plusieurs parties du projet. Ces fonctions peuvent être utilisées pour effectuer des tâches de prétraitement, de nettoyage, de transformation de données ou d'autres tâches générales que le projet peut nécessiter. Elles peuvent être appelées dans différentes parties du projet, y compris dans les fichiers de prétraitement, de modèles et de tests.
